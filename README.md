        Здравствуй, уважаемый пользователь!

  Данная программа предназначена для парсинга данных с сайта hh.ru по
компаниям и их вакансиям.
  Для примера выбраны 10 популярных компаний: Мегафон, Билайн, МТС, Yandex,
Ростелеком, Лукойл, Ozon, WILDBERRIES, Сбер, Аэрофлот.
  Чтобы добавить желаемую компанию, в файле main.py в словарь companies_dict
добавьте компанию в формате: <ключ-id компании>:<значение-название компании>.
  Перед началом работы установите postgresql на свой компьютер и настройте
БД 'postgres'(название может быть любым, но для упрощенной работы с
программой лучше оставить название по умолчанию 'postgres'. Если все же
название вашей БД отличается, в файле main.py поменяйте 'postgres' во всех
местах на свое название.).
  В папке src создайте файл database.ini с параметрами соединения с БД:
[postgresql]
host=<указать хост>
port=<указать порт>
user=<указать пользователя>
password=<указать пароль>

  Запустите файл main.py в корне проекта и следуйте инструкциям на экране!
